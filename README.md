# Stock News Predictor (Python GUI)

종목 티커를 입력하면 주식 뉴스를 RSS로 크롤링해서 수집하고,
뉴스 감성 + 최근 가격 추세를 결합해 단기 예측 지표를 보여주는 데스크톱용 Python GUI 앱입니다.

## 실행 방법

```bash
python stock_news_predictor.py
```

입력 항목
- `티커`: 예: `AAPL`, `MSFT`, `NVDA`
- `예측 영업일`: 1 ~ 60
- `뉴스 건수`: 기본 8개(최대 60)
- `가격 이력`: 가격 추세를 계산할 기간 (`1mo`, `3mo`, `6mo`, `1y`, `2y`)

앱 구성
- 현재가 / 가격 추세 / 뉴스 감성 / 예측 수익률 / 신뢰도 / 예측 구간 카드
- 최근 가격 라인 차트
- 최신 뉴스 목록 (더블 클릭 시 원문 뉴스 열기)

주의
- 가격은 Yahoo Finance 차트 API에서 조회하고, 뉴스는 공개 RSS(현재 Google News/Bing News)에서 크롤링합니다.
- 실시간 인터넷 연결이 필요합니다.
- 수치/신호는 참고용 지표이며 투자 조언이 아닙니다.
